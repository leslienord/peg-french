# Modèles

import { Callout } from 'nextra-theme-docs'

Dans cette section, nous aborderons certains modèles de langage récents et la manière dont ils appliquent avec succès les techniques de prompt engineering les plus récentes et les plus avancées. Nous aborderons également les capacités de ces modèles pour une série de tâches et de configurations de prompting telles que le few-shot prompting, le zero-shot prompting, et le chain-of-thought prompting. Il est important de comprendre ces capacités pour connaître les limites de ces modèles et savoir comment les utiliser efficacement.
